## Glossary

| Term | Definition |
|---|---|
| **Anhedonia** | Emotional numbing — reported subjective effect with BPC-157 in some users. |
| **Apoptosis** | Programmed cell death; targeted by senolytics like FOXO4-DRI. |
| **AMPK** | AMP-activated protein kinase; master metabolic regulator activated by exercise, metformin, AICAR, and MOTS-c. |
| **Bioregulator** | Short peptide (2–4 amino acids) that modulates gene expression in specific tissues; the Khavinson peptide family. |
| **Cardiolipin** | Inner-mitochondrial-membrane phospholipid stabilized by SS-31 / Elamipretide. |
| **CNPV** | FDA Commissioner's National Priority Voucher pilot — accelerated review pathway used for Orforglipron 2026 approval. |
| **COA** | Certificate of Analysis — vendor purity report, ideally HPLC + MS, batch-specific. |
| **DAC** | Drug Affinity Complex — chemical modification giving CJC-1295 DAC its 6–8-day half-life via albumin binding. |
| **DILI** | Drug-Induced Liver Injury — documented for SARMs RAD-140, LGD-4033, and Ostarine. |
| **Exerkine** | Signaling molecule released during exercise (MOTS-c, Humanin, irisin). |
| **HSDD** | Hypoactive Sexual Desire Disorder — FDA-approved indication for PT-141 (Vyleesi). |
| **HPG Axis** | Hypothalamic-Pituitary-Gonadal axis; controls testosterone and reproductive hormone production. |
| **Lipolysis** | Breakdown of stored triglyceride; targeted by HGH Frag 176-191 and AOD-9604. |
| **MDP** | Mitochondria-Derived Peptide (Humanin, MOTS-c, SS-31). |
| **Matrikine** | Matrix-derived peptide signaling cells to remodel ECM (Matrixyl, GHK-Cu). |
| **MC1R / MC4R** | Melanocortin receptors — MC1R drives tanning (Melanotan 1); MC4R drives libido and appetite (Melanotan 2, PT-141). |
| **NMPA** | National Medical Products Administration (China) — approved Mazdutide for obesity (Jun 2025) and T2DM (Sep 2025). |
| **NCT** | ClinicalTrials.gov registry identifier. Verifiable online. |
| **PEGylation** | Polyethylene-glycol conjugation that extends peptide half-life (PEG-MGF). |
| **PMID** | PubMed unique article identifier. The most authoritative reference type used in this guide. |
| **RXFP1** | Relaxin family peptide receptor 1; target of B7-33. |
| **Senescent Cell** | Aged, dysfunctional "zombie cell" that drives inflammation and aging via SASP. |
| **Senolytic** | Compound that selectively kills senescent cells (e.g., FOXO4-DRI, dasatinib + quercetin). |
| **SARM** | Selective Androgen Receptor Modulator (RAD-140, LGD-4033, Ostarine). All WADA-banned, all flagged for testosterone suppression and DILI. |
| **SNARE Complex** | Vesicle-fusion protein complex disrupted by Argireline and SNAP-8 ("topical Botox"). |
| **Tachyphylaxis** | Rapid loss of drug response with repeated dosing (e.g., kisspeptin receptor desensitization). |
| **Tyrosinase** | Melanin-producing enzyme inhibited by Decapeptide-12. |
| **VPAC1/VPAC2** | VIP receptors driving anti-inflammatory and vasodilatory cascades. |
| **WADA** | World Anti-Doping Agency. Many compounds in this guide are banned in competitive sport (TB-500, all SARMs, MOTS-c since 2024, etc.). |
